📚 Data Structure Tutor (자료구조 튜터)
Data Structure Tutor는 웹 브라우저에서 다양한 자료구조의 동작 원리를 시각적으로 학습하고 테스트해 볼 수 있는 인터랙티브 웹 애플리케이션입니다. 핵심 로직은 WebAssembly(WASM) 로 컴파일되어 웹 환경에서도 네이티브에 가까운 빠르고 부드러운 성능을 제공합니다.

✨ 주요 기능 및 화면 구성
이 프로젝트는 사용자 친화적인 3단 레이아웃으로 구성되어 있습니다.

⬅️ 좌측 패널 (자료구조 선택)

학습하고자 하는 자료구조를 선택할 수 있는 메뉴 버튼들이 위치해 있습니다.

원하는 자료구조(예: Stack, Queue, Linked List, Tree 등)를 클릭하여 중앙 화면을 전환합니다.

⏺️ 중앙 패널 (자료구조 시각화)

선택된 자료구조의 현재 상태가 그래픽으로 렌더링되는 메인 화면입니다.

데이터가 삽입되거나 삭제되는 과정, 탐색 과정 등이 시각적인 애니메이션과 함께 제공되어 내부 구조를 쉽게 이해할 수 있습니다.

➡️ 우측 패널 (관련 함수 및 조작)

현재 화면에 띄워진 자료구조를 직접 조작할 수 있는 함수 패널입니다.

해당 자료구조의 주요 연산(예: push(), pop(), insert(), delete() 등)을 버튼 클릭과 값 입력을 통해 실행해 볼 수 있습니다.

🛠️ 기술 스택 (Tech Stack)
Core Logic: WebAssembly (WASM)
